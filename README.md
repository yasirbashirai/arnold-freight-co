# Arnold Freight Co. — Website

A professional 5-page website for Arnold Freight Co., a freight and logistics company based in Chattanooga, Tennessee.

## Pages

1. **Home** — Main sales page with hero, services overview, how it works, trust stats, and quote form
2. **Services** — Detailed breakdown of freight brokerage, FTL, LTL, expedited, and dedicated support
3. **About** — Company story, core values, and why clients choose Arnold Freight
4. **Contact** — Contact information, DOT/MC numbers, quote form, and Google Maps embed
5. **Careers** — Open positions, company perks, and application form

## Tech Stack

- React 18
- Tailwind CSS 4
- React Router DOM
- React Icons

## Getting Started

```bash
npm install
npm start
```

The app runs on [http://localhost:3000](http://localhost:3000).

## Build for Production

```bash
npm run build
```

## Features

- Fully responsive design (mobile, tablet, desktop)
- Scroll-triggered animations
- Interactive quote/contact forms
- Animated statistics counter
- Dark navy + gold color scheme
- Google Maps integration on Contact page
